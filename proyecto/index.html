1. Introducción

 

Un desarrollador de software, durante el proceso de desempeño de su trabajo (desarrollo de código fuente, desarrollo de aplicaciones, etc.) , por lo general suele encontrase con cierto tipo de problemas que lo dejan atascado o algunos errores que se niega a ser resuelto, no importa lo que se haga. En casos como este, los desarrolladores de software podrían necesitar respuestas a preguntas relacionadas con varios lenguajes de programación, plataformas de desarrollo, herramientas, APIs y servicios. La gran cuestión que se plantea es ¿Dónde se podría recurrir o consultar para dar solución a estas necesidades?.

Una de las posibles soluciones podría ser el de recurrir a sitios web o comunidades de desarrolladores informáticos, de tal modo que se puedan hacer preguntas o revisar las soluciones a todas sus dudas respecto al desarrollo de software. Una de las comunidades de desarrolladores informáticos en la web más popular se denomina “Stack Overflow”.

Stack Overflow. es la comunidad de desarrolladores informáticos en línea más grande y confiable de la web, en la que estos aprenden y comparten sus conocimientos. El sitio web sirve como una plataforma para que los usuarios puedan hacer y responder preguntas. Los usuarios registrados en la comunidad y que tienen participación activa en la comunidad, pueden votar preguntas y respuestas, editar preguntas y respuestas de manera similar a un Wiki.

2. Descripción de los Datos

 

Los datos utilizados para el proyecto fueron extraídos de Stack Overflow Data , El servicio web de Google BigQuery permite realizar almacenamiento y consulta de conjuntos de datos masivos con billones de filas.

Para el proyecto se seleccionó la base de datos de Stack Overflow, esta base de datos esta conformada de 16 tablas, de los cuales las que más relevantes consideramos para el proyecto son :

Num. Tabla Descripción Atributos Relevantes
1 posts_questions Contiene información relacionada a una pregunta realizada por un usuario registrado en la comunidad
  • id
  • title
  • accepted_answer_id
  • answer_count
